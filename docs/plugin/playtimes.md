# PlayTimesについて
PlayTimesは、プレイヤーのプレイ時間を保存するPluginです。<br>
コマンドによって、表示できます。<br>

## コマンドリスト
| コマンド名 | 説明 | 権限 |
|---------|------|------|
| `/pt` | 自分のプレイ時間を表示する。 | default |
| `/pt top` | プレイ時間ランキングを表示する。 | default |
