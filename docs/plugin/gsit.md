# GSitについて
GSitは、階段やカーペット、コマンドを使用すれば地面などに座れるPluginです。<br>
また、コマンドを使用すれば、横たわったり、地面上を泳ぐことができます。

## コマンドリスト
| コマンド名 | 説明 |
|---------|------|
| `/sit` | そこに座ります。 |
| `/lay` | そこに寝ます。 |
| `/layback` | そこに寝ます。 |
| `/bellyflop` | そこに下向きに寝ます。 |
| `/spin` | その場で回ります。 |
| `/crawl` | そこに下向きで泳ぎます。 |
