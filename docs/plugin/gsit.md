# GSitについて
GSitは、階段やカーペット、コマンドを使用すれば地面などに座れるPluginです。<br>
また、コマンドを使用すれば、横たわったり、地面上を泳ぐことができます。

## コマンドリスト
| コマンド名 | 説明 | 権限 |
|---------|------|------|
| `/sit` | そこに座ります。 | default |
| `/lay` | そこに寝ます。 | default |
| `/bellyflop` | そこに下向きに寝ます。 | default |
| `/spin` | その場で回ります。 | default |
| `/crawl` | そこに下向きでクロールします。 | default |
