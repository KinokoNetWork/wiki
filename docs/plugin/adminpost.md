# AdminPostについて
運営からのお知らせを受け取るためだけの、サバイバルサーバー専用Pluginです。<br>
`/adminpost`コマンドを使用して、過去のお知らせを確認できます。<br>

## コマンドリスト
| コマンド | 説明 |
|---------|------|
| `/adminpost boxlist` | 過去のお知らせを確認します。（クリックで読めます。） |
| `/adminpost look <ID>` | 指定したIDのお知らせを確認します。 |

## 注意事項
- お知らせを読まないと、ログイン時に効果音が鳴ります。
