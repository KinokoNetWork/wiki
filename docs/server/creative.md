# クリエイティブサーバー（CreativeServer）
認証済みプレイヤーが、サバイバルサーバーで試せない回路や建築を試せるサーバーです。<br>
クリエイティブモードで、自由に建築や回路を試せます。<br>
一般プレイヤーは、アドベンチャーモードです。

---

## サバイバルとスペクテイターモードの切り替えを、一般プレイヤーができるようにしてほしい。

### A. 一般プレイヤーにクリエイティブモードやサバイバルモード権限を貸与すると、地面や建造物が荒らされる可能性があると判断しました。
