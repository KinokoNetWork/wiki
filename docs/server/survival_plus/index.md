# サバイバル＋サーバー（Survival+Server）
普通のバニラ+保護Plugin+MCMMOとかを導入したサバイバル+サーバーです！

## 通常のサバイバルサーバーと違う点
- MCMMOが導入されているので、スキルによって採掘や戦闘が有利になります。
- /homeや/tpaなどの便利コマンドが使用できるようになります。
- Griefpreventionが導入されているので、土地保護が簡単にできます。
- Discordの画像をマイクラ内で確認できるようになります。
- 資源ワールドが別で用意されています。
- その他、便利なPluginがいくつか導入されています。

## ワールド
- メインワールド：建築メイン（ワールドボーダー: 50000ブロック）
- 資源ワールド：資源採取（ワールドボーダー: 20000ブロック）
    - 資源ワールド ネザー：建築、資源採取（ワールドボーダー: 20000ブロック）
    - 資源ワールド エンド：建築、資源採取（ワールドボーダー: 20000ブロック）
