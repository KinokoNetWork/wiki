# サバイバル＋サーバー（Survival+Server）
普通のバニラ+保護Plugin+MCMMOとかを導入したサバイバル+サーバーです！<br>
参加するには、[投票](../../info/vote.md)が必要です。投票数を消費して、サーバーに参加できます。

## 通常のサバイバルサーバーと違う点
- MCMMOが導入されているので、スキルによって採掘や戦闘が有利になります。
- /homeや/tpaなどの便利コマンドが使用できるようになります。
- Griefpreventionが導入されているので、土地保護が簡単にできます。
- Discordの画像をマイクラ内で確認できるようになります。
- 資源ワールドが別で用意されています。
- その他、便利なPluginがいくつか導入されています。

# きのこサーバー Survival+サーバー ルール
すべて、KinokoNetWork利用規約に準拠し、処罰時は、こちらのルールで処罰されます。

処罰に関して気になることは、[よくある質問](../../info/qa.md)をご確認ください。

## 禁止事項
1. 他プレイヤーの建築の無断破壊、アイテム窃盗する行為
2. サーバーへ意図的に負荷を与える行為
3. 他プレイヤー・運営のなりすまし、詐欺行為
4. 自動ツール、チートクライアント、またはXray等に該当するテクスチャの使用
5. 不具合を意図的に利用、隠蔽、または他プレイヤーに示唆する行為
6. 他プレイヤーに対して、公序良俗に反する発言・誹謗中傷・差別・脅迫・強要をする行為
7. 他サーバー、他サービスへの勧誘行為
8. 他プレイヤーに対して、許可なしPvPやアイテム、ツールを使い、プレイ妨害をする行為
9. 許可なしでトラップタワーを作成しないでください。
10. その他、運営内で不適切と判断した行為

## マナー
1. 挨拶をしよう。
2. 一般常識を持とう。
3. 自分の建築物とわかるようにしよう。
4. ルール違反者がいたら、チャットで言わずに、運営に報告しよう。
5. 資源ワールドで建築してもいいですが、壊されても文句は言えません。
6. メインワールドで、資源採取はやめましょう。というかできません。
