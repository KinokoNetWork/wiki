# きのこサーバー専用 独占禁止法（ルール）
マイクラ経済サーバーで、「自由な商売」と「競争」を維持するためのルールを以下に定めます。

### 1. 「談合」の禁止
仲の良いショップ店主同士でこっそり集まって、**「明日からダイヤ1個の値段を1000円に上げようぜ」と決めるのは禁止**です。

### 2. 「不当廉売」の禁止
ライバルの店を潰すためだけに、**原価を完全に無視した安値で商売を続けること**は禁止です。

### 3. 「抱き合わせ販売」の禁止
「このレアアイテムが欲しければ、このゴミアイテムもセットで買え」と、**不必要なものを強制的に買わせること**は禁止です。

### 4. 「取引拒絶」の禁止
特定のプレイヤーやグループを仲間外れにするために、**正当な理由なく「お前には絶対売らない」と同盟を組んで無視すること**は禁止です。

### 5. 「私的独占」の禁止
サーバー内の特定の資源（シュルカーボックス、エリトラ等）を、**1つのグループが力ずくで買い占めたり、他人が入手できないよう妨害すること**は禁止です。

## ⚖️ 違反した場合のペナルティ
違反が確認された場合、以下のペナルティが科されることがあります。

- 課徴金（罰金）
- 営業停止
- その他、管理者が適切と判断する措置
