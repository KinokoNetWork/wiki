# 税金について
サバイバルサーバーでは、経済要素の一環として税金システムが導入されています。  
税金は、プレイヤーが得た収入や取引に対して課され、運営の対応費やイベント開催時に役立てられます。

## 現代社会における税金について
### 1. 税金ってなに？
税金とは、「**安心・安全・便利**」に暮らすために、みんなで出し合うお金のことです。

みんなが少しずつ出し合うことで、一人ではできない大きな仕事（道路を作ったり、警察を動かしたりすること）を実現しています。

### 2. 税金は何に使われているの？
#### 教育
* 教科書代（公立学校）
* 校舎の建設や修理
* 先生のお給料

#### 安全・救急
* 警察の運営費
* 消防の運営費
* 救急車の運営費
  * 救急車を呼んだ時に、お金を支払わなくていいのは、税金で支払われているからです。

#### 公共施設
* 道路や高速道路の建設と維持
* 公園の整備
* ゴミの収集と処理

### 3. 税金にはどんな種類があるの？
税金にはたくさんの種類がありますが、代表的なものを3つ紹介します。

| 税金の名前 | どんなときにかかるか |
| --- | --- |
| **消費税（しょうひぜい）** | お店でものを買ったり、サービスを受けたりしたとき。 |
| **所得税（しょとくぜい）** | お父さんやお母さんなど、大人が働いてお給料をもらったに、減らされるお金 |
| **住民税（じゅうみんぜい）** | 自分の住んでいる市区町村や都道府県に払うお金 |

### 4. 税金の流れ
1. **集める** 私たちが買い物や仕事をすることで、国や地方にお金を払う。
2. **計画を立てる** 政治家たちが、国会や市役所などで、「**何にいくら使うか**」を話し合って決める。
3. **使う** 道路を作ったり、お医者さんや先生を助けたりするために使われる。

## サバイバルサーバーにおける税金システム
サバイバルサーバーでは、以下のような税金システムが導入されています。

- 定期的な収入に対する所得税（10%）
