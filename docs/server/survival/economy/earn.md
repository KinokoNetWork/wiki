# どうやってお金を稼ぐか
サバイバルサーバーでは、以下の2つの方法でお金を稼ぐことができます。

1. Jobs（職業Plugin）で職に就く
2. 投票報酬で稼ぐ

## 1. Jobs（職業Plugin）で職に就く
Jobsは、サバイバルサーバーで導入されている職業システムです。  
プレイヤーは、様々な職業に就くことができ、その職業に応じた仕事をすることでお金を稼ぐことができます。  
`/jobs browse`で、職業一覧を見ることができ、右クリックで職業に就くことができます。

## 2. 投票報酬で稼ぐ
サバイバルサーバーでは、外部の投票サイトに投票することで、報酬としてお金を得ることができます。  
投票すると、250きのこいんがもらえます。
