# サンドボックスサーバー（SandBoxServer）
運営が、サンドボックス鯖以外のサーバーに導入予定・開発予定の機能を、試すためのサーバーです。<br>
基本的には、開いていませんが、大規模な機能追加の際に、サーバーが開かれます。<br>

## 参加したい場合
`[Verified]`権限を持っているユーザで、Discordも認証されていると、デバッグサーバーのチャンネルが閲覧できるようになります。
