# 鯖主とその仲間たち
このサーバーは、管理者と運営によって、管理されています。

基本的に、運営の役職は、
- Administrator
  - サーバー管理
- Builder
  - 鯖内の建築
- Developer
  - Web、Plugin開発
- Staff
  - プレイヤー監視・対応

に分けられています。

また、運営は、**ボランティア活動ということを前提に参加**していただいています。<br>
