# mcstatsについて
mcstatsは、サバイバルサーバーを中心とした、プレイヤー情報を閲覧できるサイトのことです。<br>
プレイヤーの統計情報やランキングを閲覧できます。<br>

## サイトページ
- [トップページ（`https://mcstats.kinokoroom.net/`）](https://mcstats.kinokoroom.net/)
- [プレイヤー比較（`https://mcstats.kinokoroom.net/compare`）](https://mcstats.kinokoroom.net/compare)
- [このサイトについて（`https://mcstats.kinokoroom.net/about`）](https://mcstats.kinokoroom.net/about)
- [ランキング（`https://mcstats.kinokoroom.net/lists`）](https://mcstats.kinokoroom.net/lists)
- [PvPランキング（`https://mcstats.kinokoroom.net/pvpstats`）](https://mcstats.kinokoroom.net/pvpstats)
- [マイページ（`https://mcstats.kinokoroom.net/mypage`）](https://mcstats.kinokoroom.net/mypage)

## 注意事項
- mcstatsは、サーバーのデータを元に情報を表示しています。<br>そのため、サーバーのデータが正確でない場合、mcstatsの情報も正確でない可能性があります。
- サーバーから退出しないと、データが保存されません。
