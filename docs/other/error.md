# サーバー接続エラー集
[toc]

## エラー一覧
### You must wait 0 seconds before logging-in again.
- 原因：短時間での再接続を試みた場合に発生します。
- 対処法：数秒待ってから再接続を試みてください。

### サーバー参加人数が、最高に達しています。
- 原因：サーバーの最大参加人数に達した場合に発生します。
- 対処法：しばらく待ってから再接続を試みてください。

### Outdated client! Please use {}
- 原因：クライアントのバージョンが、サーバーのバージョンと一致しない場合に発生します。
- 対処法：サーバーのバージョンに合わせて、クライアントを更新してください。

### Outdated server! I'm still on {}
- 原因：サーバーのバージョンが、クライアントのバージョンと一致しない場合に発生します。
- 対処法：サーバーの管理者に連絡して、サーバーを最新バージョンに更新してもらってください。

### Connection Timed Out
- 原因：通信環境が不安定、またはサーバーからの応答が一定時間内にない。
- 対処法：
  - サーバーが稼働しているか確認する。
  - Wi-Fiルーターの再起動、または有線LANへの切り替え。
  - PCのWindows Defenderやセキュリティソフトで、MinecraftおよびJavaの通信を許可する。
  - DNS設定を「8.8.8.8」などに変更する、またはDNSキャッシュをクリアする。

### ログインに失敗しました:無効なセッションです(ゲームとランチャーを再起動してください)
- 原因：
  - ログイン情報の有効期限が切れている、または認証サーバーに問題がある。
  - VPNやプロキシを利用している場合に発生することがあります。
- 対処法：マイクラを一度終了し、ランチャーで再ログインを行ってから再起動する。

### You are not white-listed on this server!
- 原因：サーバーのホワイトリストに登録されていない場合に発生します。
- 対処法：サーバーの管理者に連絡して、ホワイトリストに登録してもらってください。

### Authentication servers are down. Please try again later, sorry!
- 原因：Mojangの認証サーバーがダウンしている場合に発生します。
- 対処法：時間をおいてから再接続を試みてください。

### You are banned from this server.
- 原因：サーバーからのアクセスが禁止されている場合に発生します。
- 対処法：サーバーの管理者に連絡して、Ban措置の解除を依頼してください。
